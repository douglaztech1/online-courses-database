# Plataforma de Cursos Online — Projeto Fictício

Este projeto simula o banco de dados de uma plataforma de cursos online,
com foco em demonstrar fundamentos de dados relacionais e SQL básico.

## Objetivo
Modelar um banco de dados relacional e escrever consultas SQL simples
para responder perguntas comuns do negócio.

## Modelo de Dados
- Alunos podem se matricular em vários cursos.
- Cursos podem ter vários alunos (relação N–N).
- Cada curso possui várias aulas.
- O progresso dos alunos é registrado por aula.

## Tecnologias
- SQL (dialeto genérico)
- Banco relacional (conceito)
